# 🚀 coDY's Portfolio

> Personal portfolio website showcasing AI/ML projects and full-stack development skills

## 🌟 Features

- Interactive Animations with Framer Motion
- Responsive Design for all devices  
- AI/ML Project Showcase
- Modern React 18 Tech Stack
- Performance Optimized

## 🛠️ Tech Stack

- React 18, Styled Components, Framer Motion
- React Router v6, React TSParticles
- Deployed on Vercel

## 🚀 Quick Start

```bash
npm install
npm start
npm run build
```

## 📱 Sections

- Home - Interactive landing page
- About - Personal background
- Skills - AI/ML & Full-Stack expertise  
- Work - Project portfolio
- Blog - Technical articles
